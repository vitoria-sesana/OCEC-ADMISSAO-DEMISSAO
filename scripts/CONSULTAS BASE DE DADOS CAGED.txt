CONSULTAS BASE DE DADOS CAGED 

DICIONARIO CAGED ---------------
SELECT * FROM `basedosdados.br_me_caged.dicionario` WHERE id_tabela = "microdados_movimentacao" and nome_coluna = "tipo_movimentacao"


CAGED MOVIMENTACAO ANALISES ----------------
